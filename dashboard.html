🔒 Lectura bloqueada por Firestore

Firebase respondió permission-denied: las reglas actuales no permiten leer la colección encuestas. Esto en realidad es buena señal para la privacidad — pero para ver el panel hay que permitir lectura solo a un admin autenticado. Regla sugerida (confírmala con JP antes de aplicar):

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /encuestas/{doc} {
      allow create: if true;                       // la encuesta puede enviar
      allow read, update, delete: if request.auth != null;  // solo admin logueado
    }
  }
}

Mientras tanto, si quieres ver datos en una prueba rápida, puedes permitir lectura temporal — nunca dejes lectura abierta en producción con datos personales.
